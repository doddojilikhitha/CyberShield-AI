INCIDENT RESPONSE GUIDANCE BY TECHNIQUE

Ransomware Response (T1486):
1. Immediately isolate affected systems from network
2. Identify patient zero and infection vector
3. Determine ransomware variant
4. Check for backup availability
5. Engage incident response team and legal counsel
6. Do NOT pay ransom without legal consultation
7. Restore from clean backups

Credential Dumping (T1003):
1. Force password resets for all potentially affected accounts
2. Invalidate all active sessions and tokens
3. Enable MFA if not already active
4. Review privileged account activity
5. Audit LSASS protection settings

Phishing (T1566):
1. Quarantine malicious emails
2. Block sender domains and IPs
3. Search for similar emails across organization
4. Educate affected users
5. Review email gateway rules

Lateral Movement (T1021):
1. Disable compromised accounts
2. Review SMB/RDP access logs
3. Segment network to contain spread
4. Reset service account passwords
5. Audit privileged access
