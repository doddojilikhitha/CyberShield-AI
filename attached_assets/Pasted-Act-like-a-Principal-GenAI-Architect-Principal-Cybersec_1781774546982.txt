USER WORKFLOW

1. Analyst submits incident description.
2. Incident Classifier Agent analyzes incident.
3. Framework Mapper Agent maps incident to:
   - NIST
   - MITRE ATT&CK
   - OWASP
4. RAG Retrieval Agent retrieves supporting guidance.
5. Playbook Generator Agent creates response playbook.
6. Compliance Reviewer Agent validates output.
7. Human Review stage.
8. Analyst can:
   - Approve
   - Reject
9. If rejected:
   - Feedback is captured
   - Playbook Regeneration Agent creates improved version
10. Approved playbooks are stored.
11. Report generation is available.
